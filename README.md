# Famous quotes
Is an app that shows famous quotes, where you can click the quote thinker and see his/ her famous quotes. 
> Developed a calculator App  using react-redux
> This app uses ReactRouters
>Tests for fetching data where done.
> To run the project clone this repo install dependencies `npm install` and run the app `npm start`

## Built With

- Html, Js, Css.
- Webpack.
- React
- Redux
- Jest
- React Testing Library
- Linters.
- ReactRouter
